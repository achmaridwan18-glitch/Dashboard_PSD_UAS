# 🌿 Dashboard Produksi Tanaman Perkebunan Indonesia

Dashboard interaktif berbasis **Streamlit** untuk menganalisis data produksi tanaman perkebunan di Indonesia tahun 2025. Dibuat sebagai proyek UAS Data Science.

![Python](https://img.shields.io/badge/Python-3.9+-blue?logo=python)
![Streamlit](https://img.shields.io/badge/Streamlit-1.37-red?logo=streamlit)
![License](https://img.shields.io/badge/License-MIT-green)

---

## ✨ Fitur Utama

### 📊 Visualisasi Data
- **KPI Cards**: Total produksi, jumlah provinsi, komoditas, rata-rata
- **Bar Chart**: Produksi per komoditas
- **Pie Chart**: Distribusi produksi (Top 8)
- **Line Chart**: Produksi per provinsi
- **Histogram**: Distribusi produksi
- **Boxplot**: Sebaran produksi per komoditas
- **Scatter Plot**: Luas area vs produksi
- **Heatmap**: Korelasi antar variabel numerik

### 🏆 Analisis Ranking
- Top 10 Provinsi penghasil terbesar
- Ranking komoditas unggulan

### 🤖 Machine Learning
- **Random Forest Regressor** dengan feature importance
- **Linear Regression** dengan pipeline scaling
- Evaluasi model: MAE, RMSE, R² Score
- Actual vs Predicted visualization

### 💡 Insight Otomatis
- Komoditas unggulan
- Provinsi terproduktif
- Rekomendasi kebijakan

### 📥 Download
- Export data filter (CSV & Excel)
- Export ringkasan statistik
